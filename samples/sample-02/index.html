CSV Data Cleaner

Remove duplicate rows, strip stray whitespace, and standardise column names to snake_case — then get a clean, ready-to-use CSV back.

Subscription required — This sample runs inside Claude.ai using the Cowork feature. A Pro or Team subscription with Cowork access is required. No API key or local Python install needed.

What this does

Messy CSVs often come with inconsistent column names (spaces, CamelCase), extra whitespace in cells, and accidentally duplicated rows. This task asks Claude to audit your file, report every issue it finds, and output a fully cleaned version with:

  • Column names converted to snake_case
  • Leading/trailing whitespace removed from all cells
  • Exact duplicate rows removed

Step-by-step in Claude Cowork

  1. Open claude.ai and sign in to your Pro or Team account.
  2. Click Cowork in the left sidebar to open a Cowork session.
  3. Upload your CSV file using the attach / upload button. For a quick test, use sample_data/messy.csv from this repo.
  4. Paste the prompt below into the chat and press Enter: 
    I've uploaded a CSV file that needs cleaning. Please:

1. Read the CSV and show me its column names and the first 5 rows.
2. Rename all column headers to snake_case (lowercase, spaces and hyphens replaced with underscores, CamelCase split).
3. Strip leading and trailing whitespace from every cell value.
4. Remove any exact duplicate rows (all fields identical).
5. Report a summary: how many columns were renamed, how many cells had whitespace stripped, how many duplicate rows were removed, and how many rows remain.
6. Output the full cleaned CSV as a code block (using triple backticks with "csv" language tag) so I can copy and save it.

Be thorough — show me what each column name changed from and to.
  5. Claude will display the before/after column names, a cleaning summary, and the full cleaned CSV you can copy.
  6. Copy the CSV from the code block and save it as a new file. Ask Claude for any further tweaks if needed.

Reference: local Python version

The Python script main.py in this folder performs the same cleaning locally using only the standard library:

# Clean messy.csv and write clean_messy.csv:
python main.py sample_data/messy.csv

# Specify a custom output path:
python main.py sample_data/messy.csv --output cleaned.csv
Note: The Python script requires local file access and Python 3.8+. The Cowork approach above works entirely in the browser with no install required.
