Your task

Build a Claude Cowork automation that generates a Python script to fetch data from an API or URL and save it to a local file. You choose the data source and output format, then write a prompt that makes Claude produce a working script.

  • (Optional) A screenshot of the API docs, or paste the URL and a sample JSON response directly in your prompt

Your Cowork prompt

Things to decide before writing:
• What is the API URL or data source?
• Does it need authentication? (API key in a header? query param?)
• What fields do you want to extract from the response?
• Output format: JSON, CSV, or Markdown?
• Should the script run once or poll on an interval?
• Should output files be timestamped so they don't overwrite each other?

How to run it

  1. Open claude.ai and sign in to your Pro or Team account.
  2. Click Cowork in the left sidebar.
  3. Upload API docs or paste a sample response (optional but helpful).
  4. Paste your completed prompt from cowork-prompt.md.
  5. Claude will generate a Python script — copy it into main.py and run it locally.

Bonus: local Python script

The file main.py in this folder is a skeleton. Paste Claude's generated script into it and run:

python main.py --url https://your-api-url --format json
