Smart Folder Sorter

Automatically organise a messy folder by sorting files into subfolders by type — Images, Documents, Videos, Audio, Code, Archives, and Other.

Subscription required — This sample runs inside Claude.ai using the Cowork feature. A Pro or Team subscription with Cowork access is required. No API key or local Python install needed.

What this does

Given a messy folder of mixed files, this task groups every file by its extension into category subfolders: Images, Documents, Videos, Audio, Code, Archives, and Other. It reports exactly what it would move — you review before anything changes.

Step-by-step in Claude Cowork

  1. Open claude.ai and sign in to your Pro or Team account.
  2. Click Cowork in the left sidebar to open a Cowork session.
  3. Upload the files from your messy folder using the attach / upload button. You can select multiple files at once. (For a quick test, use the files in sample_data/ from this repo.)
  4. Paste the prompt below into the chat and press Enter: 
    I've uploaded a set of files from a mixed folder. Please analyse each file by its name and extension, then group them into these categories:

- Images: .jpg .jpeg .png .gif .bmp .svg .webp .tiff .ico
- Documents: .pdf .doc .docx .txt .xls .xlsx .ppt .pptx .odt .rtf .md
- Videos: .mp4 .avi .mov .mkv .wmv .flv .webm .m4v
- Audio: .mp3 .wav .flac .aac .ogg .m4a
- Code: .py .js .ts .html .css .java .cpp .c .h .go .rs .rb .php .sh .json .xml .yaml .yml .toml .ini
- Archives: .zip .tar .gz .rar .7z .bz2 .xz .tgz
- Other: everything else

For each category, list the filenames. Then produce a summary table showing how many files fall into each category.

Finally, write out the shell commands I would need to run (on macOS/Linux) to actually move each file into a subfolder named after its category.
  5. Claude will list every file, assign categories, show a summary table, and generate ready-to-run shell move commands.
  6. Review the plan, then copy and run the shell commands in your terminal — or ask Claude to adjust the categories first.

Sample output

After pasting the prompt you should see something like:

Here's how I'd sort the uploaded files:

**Images (1)**  logo.png

**Documents (2)**  budget.xlsx, notes.txt

**Videos (1)**  video.mp4

**Code (1)**  script.py

**Other (0)**  —

---
| Category  | Count |
|-----------|-------|
| Images    |   1   |
| Documents |   2   |
| Videos    |   1   |
| Code      |   1   |

Shell commands to move files:
mkdir -p Images Documents Videos Code
mv logo.png Images/
mv budget.xlsx notes.txt Documents/
mv video.mp4 Videos/
mv script.py Code/

Reference: local Python version

The Python script main.py in this folder does the same job locally, using only the standard library. Run it if you have Python 3.8+ installed:

# Preview (dry run — no files moved):
python main.py sample_data/ --dry-run

# Actually move files:
python main.py sample_data/
Note: The Python script requires local file access and Python 3.8+. The Cowork approach above works entirely in the browser with no install required.
