CSV Summary Report Generator

Upload any CSV and get a complete statistical summary — row counts, min/max/mean for numeric columns, and top values for text columns — as a formatted Markdown report.

Subscription required — This sample runs inside Claude.ai using the Cowork feature. A Pro or Team subscription with Cowork access is required. No API key or local Python install needed.

What this does

Rather than opening a CSV in a spreadsheet and manually calculating statistics, this task hands the file to Claude and gets back a structured Markdown report covering:

  • Overview — row count, column count, column list
  • Numeric columns — count, min, max, mean, median, sum, and empty-cell count
  • Text columns — unique value count, top-5 most frequent values with their frequencies
  • Empty columns — flagged for removal

The report is output as a Markdown code block you can copy directly into a README, Notion page, or any document.

Step-by-step in Claude Cowork

  1. Open claude.ai and sign in to your Pro or Team account.
  2. Click Cowork in the left sidebar to open a Cowork session.
  3. Upload your CSV file using the attach / upload button. For a quick test, use sample_data/sales.csv from this repo.
  4. Paste the prompt below into the chat and press Enter: 
    I've uploaded a CSV file. Please generate a full statistical summary report in Markdown format.

The report must include:

## Overview
A table with: filename, total rows, total columns, and all column names.

## Numeric Columns
For each column where ≥80% of non-empty values are numbers, produce a table row with:
count, min, max, mean (2 decimal places), median (2 decimal places), sum, empty cell count.

## Text Columns
For each text/categorical column, show:
- Non-empty count
- Empty cell count
- Number of unique values
- Top 5 most frequent values and their counts (as a small table)

## Empty Columns
List any columns where all values are empty.

Output the entire report as a single Markdown code block (triple backticks with "markdown" tag) so I can copy and paste it.
  5. Claude will analyse the CSV and output the complete report inside a Markdown code block.
  6. Copy the Markdown and paste it wherever you need it — or ask Claude follow-up questions about the data, e.g. "Which product had the highest total sales?"

Sample output

```markdown
# CSV Summary Report: `sales.csv`

## Overview

| Property     | Value                                  |
|--------------|----------------------------------------|
| File         | sales.csv                              |
| Rows         | 150                                    |
| Columns      | 5                                      |
| Column names | date, region, product, units, revenue  |

## Numeric Columns (2)

| Column  | Count | Min  | Max     | Mean    | Median | Sum      | Empty |
|---------|-------|------|---------|---------|--------|----------|-------|
| units   | 150   | 1    | 500     | 87.34   | 72.00  | 13101    | 0     |
| revenue | 148   | 9.99 | 4999.00 | 612.45  | 489.50 | 90642.60 | 2     |

## Text Columns (3)

### `region`
- Count: 150 | Empty: 0 | Unique: 4
- Top 5: North (52), South (41), East (33), West (24)

### `product`
- Count: 150 | Empty: 0 | Unique: 8
- Top 5: Widget A (38), Widget B (29), Gadget X (27), ...
```

Reference: local Python version

The Python script main.py in this folder generates the same Markdown report locally using only the standard library:

# Generate report from sales.csv:
python main.py sample_data/sales.csv

# Save report to a custom path:
python main.py sample_data/sales.csv --output reports/sales_summary.md
Note: The Python script requires local file access and Python 3.8+. The Cowork approach above works entirely in the browser with no install required.
